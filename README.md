# Velora

Velora is a Streamlit web app that opens on a landing page, accepts a CSV upload, and then turns that file into an interactive analysis workspace.

## Features

- Front page with upload-first flow
- Automatic schema and column type detection
- Data quality summary
- Numeric distributions and scatter comparisons
- Categorical breakdowns
- Datetime trend analysis when possible
- Raw data preview and download

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
